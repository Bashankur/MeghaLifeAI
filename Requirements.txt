MeghaLifeAI - Project Requirements
# ---------------------------------
# This file lists the core tools, frameworks, and libraries
# required to build and run the MeghaLifeAI Android application.

# Development Environment
Android Studio Giraffe or later
Android SDK 33+
Gradle 8+

# Programming Language
Kotlin 1.9+

# UI Framework
Jetpack Compose
Material 3

# AI / Machine Learning
TensorFlow Lite
TensorFlow Lite Support Library

# Maps & Location
Google Maps SDK for Android
Google Play Services Location

# Backend / Services
Firebase Authentication (Anonymous Login)

# Architecture & Utilities
AndroidX Libraries
Kotlin Coroutines

# Build & Dependency Management
Gradle (Android Plugin)

# Target Platform
Android 10 (API 29) and above
